## Description
The project's purpose is building a ecommerce website for selling goods. Using pure PHP to build a ecommerce website. I also used pure HTML, CSS, Javascipt to build UI for the web.

1. Enviroment:  
	nginx >= 1.19.10
	mariadb >= 5.7.33
	PHP >= 7.4
2. DB:
	db.sql
4. Config file:  
	configs.php
	nginx.conf
5. Setup package:  
	composer dump-autoload
